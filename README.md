# 智慧校園二手共享系統

This repository implements the requested RWD + a11y strengthened plan as a starter system:

- Frontend: Next.js App Router + Tailwind CSS.
- Admin: pure PHP + Bootstrap + Chart.js.
- Database: MySQL/MariaDB schema shared by frontend and admin.
- Captcha dependency: Gregwar/Captcha via Composer.

## Structure

- `app/`: Next.js frontend routes.
- `components/`: responsive and accessible frontend UI components.
- `lib/`: frontend mock data and helpers.
- `admin/`: pure PHP admin dashboard.
- `database/schema.sql`: MySQL/MariaDB schema.
- `database/seed.sql`: starter categories and placeholder users.
- `docs/acceptance-checklist.md`: RWD, a11y, and functional acceptance checks.

## Frontend

Install dependencies and run:

```bash
npm install
npm run dev
```

The frontend is mobile-first and supports tablet and desktop layouts:

- Mobile uses bottom navigation and compact two-column listing cards.
- Tablet uses wider grids.
- Desktop uses persistent filtering and constrained content width.

## PHP Admin

Install PHP dependencies:

```bash
composer install
```

Import the database:

```bash
mysql -u root -p < database/schema.sql
mysql -u root -p < database/seed.sql
```

Then update `admin/includes/config.php` with your database credentials.

Or copy `.env.example` to `.env` and update the database values there. The PHP admin reads `DB_HOST`, `DB_NAME`, `DB_USER`, `DB_PASS`, and `ADMIN_SESSION_NAME` from `.env`.

Important: replace placeholder password hashes in `database/seed.sql` with real values generated by `password_hash()`.

## Admin Security

The admin module uses:

- Dedicated admin session name.
- `password_verify()` for login.
- `session_regenerate_id(true)` after login.
- CSRF token validation for POST actions.
- PDO prepared statements.
- `htmlspecialchars()` output escaping.
- `admin_logs` audit rows for sensitive changes.

## Verification

Recommended checks:

```bash
npm run typecheck
npm run build
```

Then run through `docs/acceptance-checklist.md` in mobile, tablet, and desktop viewports.
